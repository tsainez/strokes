## 2024-05-22 - Initial Setup
**Learning:** This repo contains a hybrid iOS/Web architecture. The web portion uses HanziWriter for stroke practice.
**Action:** Focus on web-based UX improvements in `Strokes/web` as they are testable and impactful.

## 2024-05-22 - Quiz Pacing
**Learning:** Immediate resetting of the quiz canvas prevents users from seeing their completed work and feeling a sense of accomplishment. It also feels jarring.
**Action:** Always add a small delay (e.g., 1 second) before resetting interactive elements upon completion to allow for processing and satisfaction.
